Installation
============

Install using pip
::
  $ pip install git+https://github.com/rdyro/sensitivity_jax.git
or from source
::
  $ git clone git@github.com:rdyro/sensitivity_jax.git
  $ cd sensitivity_jax
  $ python3 setup.py install --user

Testing
-------

Run all unit tests using
::
  $ python3 setup.py test
